# LOVORA

LOVORA is a premium AI character platform built with Next.js, React, TypeScript, Supabase, and OpenAI.

## Features

- Discover built-in and public AI characters
- Create and manage custom characters
- Share public character pages
- Character studio workflow
- Modern app shell and landing experience

## Tech Stack

- Next.js
- React
- TypeScript
- Supabase
- OpenAI
- Tailwind CSS

## Getting Started

```bash
npm install
npm run dev
